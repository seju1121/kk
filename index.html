DaeguWar

서버 소개

대구전쟁은 치열한 전투와 전략이 어우러진 마인크래프트 서버입니다. 친구들과 함께 대구를 배경으로 한 전쟁의 세계를 탐험하세요!

서버 주요 특징

  • 실시간 PVP 모드: 다른 플레이어와 실력을 겨뤄보세요.
  • 다양한 미션과 퀘스트: 새로운 도전에 도전하세요.
  • 커스터마이징 가능한 건축물: 창의력을 발휘하세요.

서버 규칙

게임의 공정성을 위해 아래의 규칙을 준수해주세요:

  1. 모든 플레이어에게 예의를 지키세요.
  2. 부정행위(해킹, 치트 등)를 삼가세요.
  3. 운영진의 지시에 협조해주세요.
  4. 팀 플레이에서 협력과 공정을 유지하세요.

참여 방법

서버에 접속하려면 아래 단계를 따르세요:

  1. 마인크래프트 클라이언트를 실행합니다.
  2. 서버 주소: daegu-war.net를 입력합니다.
  3. 접속 후 모험을 시작하세요!
